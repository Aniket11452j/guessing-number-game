# 🎯 Guessing Number Game

A fun and interactive Python game where the player tries to guess a randomly generated number with hints and score tracking.

---

## 🚀 Features

* 🎲 Random number generation
* 📊 Difficulty levels (Easy / Medium / Hard)
* 🧠 Smart hints (Too high / Too low)
* 🏆 High score tracking
* 📈 Leaderboard system
* 💾 Data stored using JSON files

---

## 🛠️ Tech Stack

* Python 🐍
* JSON (for storing scores and leaderboard)

---

## 📂 Project Structure

```
guessing-number-game/
│── game.py

```

---

## ▶️ How to Run

1. Install Python (if not installed)
2. Clone this repository:

   ```
   git clone https://github.com/Aniket11452j/guessing-number-game.git
   ```
3. Navigate to the project folder:

   ```
   cd guessing-number-game
   ```
4. Run the game:

   ```
   python game.py
   ```

---

## 🎮 How to Play

* Choose a difficulty level
* Guess the number within the range
* Get hints after each guess
* Try to beat the high score!
